A Little About Me

I'm a technology enthusiast. Whether it's trying a new Linux distro, discovering new forms of task automation, or learning a new JavaScript feature or framework, I find technology endlessly fascinating. Technology evolves at such a fast rate that there's always something new to learn!

Technology has been my hobby since high school (it started when I googled "how to program" after watching the original Tron movie). However, within the US school system a career in technology was never presented as something for me, and as such I didn't choose to pursue it within the scope of traditional education. After graduating from UC Santa Barbara with a degree in Biological Anthropology I still found myself with an interest in tech and a strong desire to continue learning. Now, after 10 months and countless hours of self-teaching, web development has grown from a fun hobby to my career choice.

Here's what I can do

With such a large and constantly growing selection of technology, choosing the right tool for the job is a skill all on it's own. It's all too easy to learn one framework and use it for everything, even if the project falls outside of the framework's ideal use case. For example, this website was built using only HTML and CSS. No JavaScript, no Bootstrap, no extra libraries (you can check out the source here!). While I would have had just as much fun using React or a CSS library, that would have been unnecessary weight for this simple static website.

Like this website, each of the following projects was designed to use the technology that best fit its use case, with little else included that might distract from the project's core functionality.

Todo List with React

Fetches todo data from jsonplaceholder.com while allowing for new user added items. Manages state using React.

Gradient Generator

The user can pick two different colors or have two randomly selected. The colors are then displayed as a gradient in the background. The css needed to create the effect is displayed below. Uses vanilla JavaScript for DOM manipulation.

Database Search

Fetches and displays a user database from jsonplaceholder.com. Robot avatars are created from each user's id number using robohash.org The user can search within the database and results will be instantly displayed. Uses React.
