A Little About Me

I'm a technology enthusiast. Whether it's trying a new Linux distro, discovering new forms of task automation, or learning a new JavaScript feature or framework, I find technology endlessly fascinating. Technology evolves at such a fast rate that there's always something new to learn!

Software has been my hobby since high school (it started when I googled "how to program" after watching the original Tron movie). Within the US school system a career in technology was never presented as something for me, and as a result I never choose to pursue it within the scope of traditional education. However, after graduating from UC Santa Barbara with a degree in Biological Anthropology I still found myself with an interest in tech and a strong desire to continue learning. Now after 10 months and many hours of self-teaching, web development has grown from a fun hobby to my career choice.

Here's what I can do

I love looking at other people's projects, so please feel free to check out what I've built! As a fullstack JavaScript developer there is no shortage of new tools, so when building a project I focus on 'choosing the right tool for the job.'

For example, if a project didn't need to manage a global state with React, then good ol' vanila JavaScript was all that was used. (This website follows the same design! No CSS or JavaScript libraries were used. You can check out the source here!)

Todo List with React

Fetches todo data from jsonplaceholder.com while allowing for new user added items. Manages state using React.

Gradient Generator

The user can pick two different colors or have two randomly selected. The colors are then displayed as a gradient in the background. The css needed to create the effect is displayed below. Uses vanilla JavaScript for DOM manipulation.

Database Search

Fetches and displays a user database from jsonplaceholder.com. Robot avatars are created from each user's id number using robohash.org The user can search within the database and results will be instantly displayed. Uses React.
