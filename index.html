A Little About Me

I'm a technology enthusiast. Whether it's trying a new Linux distro, building new forms of task automation, or learning new JavaScript features or frameworks, I find technology endlessly fascinating.

While it is a clich√© story, software has been my hobby since high school (it started when I googled "how to program" after watching the original Tron movie). Despite my interest, professional programming always seemed like something out of my reach. While in school, a career in technology was never presented as a possibility, instead it looked like that path was reserved for those who "loved math." An enthusiasm for programming was not enough it seemed.

As a result, I didn't choose to pursue programming within traditional education. However, after graduating from UC Santa Barbara with a degree in Biological Anthropology, I still found myself with an interest in tech and a desire to continue learning. Now after 10 months and many hours of self-teaching, tinkering, and building, web development has grown from a fun hobby to my career choice.

Here's what I can do

Being a self-taught developer, it made the most sense to start learning the most popular languages and frameworks. Thanks to the large community around JavaScript development there was no shortage of information, and getting up to speed on current trends was not difficult. I am now comfortable creating projects using HTML, CSS, and JavaScript (including all that good ES6 syntax and ES8 async/await), and am currently learning React, Node, and Express.

With so many available technologies it can be easy to overcomplicate a project. I was tempted to build this site with React, but without a state to manage or the need to dynamically redraw the DOM, it would have been an unnecessary over complication. My current focus in development is building the most efficient project possible, which I hope to have demonstrated in the projects below! You can also check out the source to this website.

Todo List with React

Fetches todo data from jsonplaceholder.com while allowing for new user added items. Manages state using React.

Gradient Generator

The user can pick two different colors or have two randomly selected. The colors are then displayed as a gradient in the background. The css needed to create the effect is displayed below. Uses vanilla JavaScript for DOM manipulation.

Database Search

Fetches and displays a user database from jsonplaceholder.com. Robot avatars are created from each user's id number using robohash.org The user can search within the database and results will be instantly displayed. Uses React.
