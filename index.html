A Little About Me

I'm a technology enthusiast. Whether it's trying a new Linux distro, building new forms of task automation, or learning new JavaScript features or frameworks, I find technology endlessly fascinating.

While it is a clich√© story, software has been my hobby since high school (it started when I googled "how to program" after watching the original Tron movie). Despite my interest, professional programming always seemed like something out of my reach. While in school, a career in technology was never presented as a possibility, instead it looked like that path was reserved for those who "loved math." An enthusiasm for programming was not enough it seemed.

As a result, I didn't choose to pursue programming within traditional education. However, after graduating from UC Santa Barbara with a degree in Biological Anthropology, I still found myself with an interest in tech and a desire to continue learning. Now after 11 months and many hours of self-teaching, tinkering, and building, web development has grown from a fun hobby to my career choice.

Project Links

Thanks to the amazing community around web development there is an amazing amount of information available to the self-taught developer. I am now comfortable creating projects using HTML, CSS, and JavaScript (including all that good ES6 syntax and ES8 async/await), and am currently learning React, Node, and Express.

As an exercise in furthering my understanding of Vanilla JavaScript, HTML, and CSS, I have written this page without the use of any libraries. You can check out the source here! For other examples of my work please take a look at the projects below.

Clarifai Face Recognition

An image url is submitted by the user, then, using the Clarifai Face Recognition API, the image is returned with a box outlining the detected face. Uses React.

Todo List with React

Fetches todo data from jsonplaceholder.com while allowing for new user added items. Manages state using React.

Star Wars API Frontend.

Frontend designed to fetch data from the Star Wars API. Searches the database for a user-submitted character name and parses the returned JSON data into a human-readable (and grammatically correct) list of movie appearances.

Gradient Generator

The user can pick two different colors or have two randomly selected. The colors are then displayed as a gradient in the background. The css needed to create the effect is displayed below. Uses vanilla JavaScript for DOM manipulation.

Database Search

Fetches and displays a user database from jsonplaceholder.com. Robot avatars are created from each user's id number using robohash.org The user can search within the database and results will be instantly displayed. Uses React.
